4Pet Description

  • Topic: Animal based Social Website

  • Technology: React.js(Front-end), firebase(Back-end)

  • Role of Project: Back-end developers, Tester

  • Project Purpose: Senior Project


    • • Developed a social-based website that enables network among local people with the same pets.

    • Uses MCV software architectural patterns and is designed to utilize express controllers and routes.

    • Controllers and paths can be used to simplify how and how parameters are passed to the method in a fast and concise way for the front end.

    • Front-end can forward URLs with the required parameters to the backend and receive the required information in a consistent manner.

    • Stored users' information in the Firebase database, and when requested by the backend, the appropriate users and their information were referenced to provide a simple way to store and retrieve information.

    • Able to a use chat feature created using the UI kit that came with the Cometchat API for quick feature creation and deployment
